Full information about the system

  • The site is intended solely for members of the Federation.

    This is justified by the fact that the system stores information about coaches and athletes of the Federation. Each member has a pass or a subscription on which his / her card number is indicated.

  • Enter data correctly.

    This is due to the fact that so far the system does not involve independent changes in data.

  • Please note that when registering, you must specify your type: coach or athlete to continue to correctly enter information.

  • On the main page you can find information about the user you are interested in by his handle.

    Only registered users will be found.

  • In the Coach tab, which is located in the top menu, you can find all coaches registered in the system. Also on the left is information.

  • In the Athlete tab you can find athletes registered in the system. And on the left information about the upcoming competitions.
